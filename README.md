📊 Marketing Analysis Project
🚀 Overview

This project turns raw marketing data into meaningful insights that can support better business decisions. Using SQL, the data is cleaned and transformed into a structured format. Python is then used to perform advanced sentiment analysis on customer feedback, helping uncover how customers truly feel. Finally, Power BI brings everything together through interactive dashboards that make the insights easy to understand and explore.

🛠️ Tools Used
SQL – Data cleaning and transformation
Python – Sentiment analysis and deeper insights
Power BI – Interactive dashboards and visualization
🔄 Workflow
Prepare and transform raw data using SQL
Analyze customer sentiment using Python
Build interactive dashboards in Power BI
Present insights using clear data storytelling
📊 What This Project Shows
How customer behavior impacts marketing performance
Which campaigns perform best
Insights from customer feedback using sentiment analysis
Clear visual trends through dashboards
🎯 Purpose

The goal of this project is to demonstrate how different tools can be combined to create a complete data analysis workflow. It focuses not just on analysis, but on storytelling—turning data into insights that are easy to understand and useful for decision-making.
